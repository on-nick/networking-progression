## Objective

Understand how data moves through the network stack.

## Concepts Learned

- OSI Model
- TCP/IP Model
- Encapsulation
- Decapsulation

## OSI Layers

1. Physical
2. Data Link
3. Network
4. Transport
5. Session
6. Presentation
7. Application

## TCP/IP Layers

- Application
- Transport
- Internet
- Network Access

## Lab

Created a simple LAN using two PCs and one switch.

## Observations

Simulation mode showed:
- ARP packets
- ICMP packets

## Lessons Learned

Data is wrapped with headers at each layer before transmission.
