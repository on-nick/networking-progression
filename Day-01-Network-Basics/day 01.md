
# Day 01 - Network Basics

## Objective

Understand how two computers communicate in a network.

## Concepts Learned

- Network
- Switch
- IP Address
- Ping
- Packet

## Lab

Topology:

PC0 ---- Switch ---- PC1

IPs:

PC0 = 192.168.1.10

PC1 = 192.168.1.11

## Verification

Successfully pinged between both PCs.


## Lessons Learned

Devices on the same network can communicate through a switch.
