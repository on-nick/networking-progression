# Day 03 - IP Addressing

## Objective

Understand IPv4 addressing and network communication.

## Concepts Learned

- IPv4
- Public IP
- Private IP
- Subnet Mask
- Loopback Address

## Commands Used

```bash
ip a
ip route
hostname -I
ping 8.8.8.8
```

## Lab

Built a network using:

- 2 PCs
- 1 Switch

## Observations

Devices communicated successfully when they were in the same subnet.

Communication failed when devices were placed in different subnets without a router.

## Lessons Learned

Every device requires a unique IP address.
