#  Networking Journey 
This repository documents my hands-on journey to learn computer networking using Packet Tracer, Wireshark, and Linux tools.

## 🎯 Goals
- Understand networking from basics to enterprise level
- Build real-world network topologies
- Learn routing, switching, VLANs, security
- Develop troubleshooting skills

## 🛠 Tools Used
- Cisco Packet Tracer
- Wireshark
- Kali Linux
- Nmap

## 📅 Progress
- Day 1–7: Networking fundamentals
- Day 8–14: Switching & VLANs
- Day 15–21: Routing & DHCP
- Day 22–30: Security & enterprise design

## 📂 Final Project
- Multi-VLAN enterprise network
- OSPF routing
- DHCP + NAT + ACL configured

## 📌 Key Skills Learned
- Network design
- Troubleshooting
- Packet analysis
- Router & switch configuration
